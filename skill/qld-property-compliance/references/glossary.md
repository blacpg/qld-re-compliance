# Glossary

Plain-English terms used across the rules. These are working definitions for agents, not statutory definitions. Where a statutory definition governs, the rule cites it.

- **Appointment (Form 6)** — the written appointment of a property agent by a client, using the approved Form 6, before the agent performs services.
- **Client** — the person who appoints the agent (typically the seller).
- **Commission** — the agent's fee for the service, as agreed in the appointment.
- **OFT** — Queensland Office of Fair Trading, the regulator for property agents.
- **POA** — Property Occupations Act 2014 (Qld).
- **PO Regulation** — Property Occupations Regulation 2014 (Qld).
- **PLA 2023** — Property Law Act 2023 (Qld), which contains the seller disclosure scheme that commenced 1 August 2025.
- **Form 2** — the approved Seller Disclosure Statement under the seller disclosure scheme.
- **Form 8** — buyer-side disclosure form, where relevant.
- **Seller disclosure** — the obligation on a seller to give the buyer prescribed information before the buyer signs the contract.
- **Authority tier** — how much weight a statement carries: legislation, regulation, statutory_form, regulator_guidance, industry_commentary, internal_policy.
- **verification_status** — how well a rule is sourced: verified, partially_verified, unverified, needs_review.
- **risk_level** — how serious it is if the agent gets the rule wrong: high, medium, low.
